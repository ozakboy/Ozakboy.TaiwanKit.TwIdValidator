# 更新紀錄

**Ozakboy.TaiwanKit.TwIdValidator** 的所有重要變更記錄於此。

## [1.0.0] - 2026-07-04

### 新增功能

- 台灣身分證驗證，涵蓋三種證號並自動辨識：
  - 本國人身分證字號（第 2 碼 1/2）
  - 新式外來人口統一證號，2021 年起發行（第 2 碼 8/9）
  - 舊式外僑居留證統一證號（前兩碼為英文字母）
- 詳細驗證結果（`TwIdResult`）：證號類型、性別、戶籍縣市名稱（已停用代碼附標注）、失敗原因
- 統一編號驗證，含財政部 2021 年 4 月新制檢核（5 的倍數）與第 7 碼為 7 的特例
- 手機號碼驗證（嚴格 `09xxxxxxxx`）與 `TryNormalize` 正規化：支援 `+886`/`886` 國碼、空格、dash、括號、句點與全形數字
- 三類號碼的測試假資料產生器（`Generate`），產生檢核碼合法的樣本；可指定 seed 重現
- 全公開 API 中英雙語 XML 文件註解

### 技術改進

- 目標框架：`netstandard2.0` / `netstandard2.1` / `net8.0` / `net9.0` / `net10.0`
- 零執行期依賴；純函式、線程安全
- 可重現建置（Deterministic）、SourceLink、符號套件（`snupkg`）、XML 文件
